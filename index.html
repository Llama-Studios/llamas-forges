Llama's Forges allows you to mix tool/armor materials make cool gear. For example, instead of just a diamond pickaxe, you can make a diamond pickaxe with an iron rod. There are three main machines in this mod:

The Melting Basin

The melting basin is used to make molten ore.

Fueling

The melting basin requires lava as fuel. Interact (right click) on it with a lava bucket to put the lava in. After each time you use it, there is a 10% chance it will need more lava, indicated by the bar on the side.

Melting

Interact (right click) on it with an ore to melt the ore. You can also melt lava to make cobblestone. The molten ore can be removed using a bucket.

Zinc

Zinc is a new greenish-colored ore. It is used mainly to make brass, but also to craft the wrench and the template table. As a bonus, zinc nuggets can be made into white dye, as zinc is an ingredient in white paint in real life.

Alloying

The melting basin can also make alloys. If you place two ores in it that can be alloyed, they will mix. For example:
Gold + Debris = Netherite
Water + Lava = Obsidian
Zinc + Gold* = Brass

*I know brass is really made using copper, not gold. I will change this once 1.17 comes out

Removing Machines

All the machines in this mod except the template table can be removed using the wrench. Sneak-interact (shift-right click) using the wrench to remove the machine. The template table can be broken like a normal block.

The Template Table

The template table is used to make templates, which are used to make molds. To use it, simply right click/tap it, and make the template you would like. For tool rods, there is no template, instead use a stick.

The Casting Basin

The casting basin is used to make molds, and make casts from the molds.

Making Molds

First, interact (right click) using the template you would like to create a mold for. Then, interact (right click) with either a clay block for a clay mold, or a bucket of molten brass for a brass mold. Clay molds are cheaper, but brass molds are reusable. Sneak-interact (shift-right click) on it to remove the mold.

Using Molds

First, interact (right click) using the mold to place it in the casting basin. Then, interact (right click) with a bucket of molten ore to make a cast. For example, to make a diamond pickaxe head, use a pickaxe mold and a bucket of molten diamond. Sneak-interact (shift-right click) on it to remove the mold when you are done.

The Assembly Table

The assembly table is used to put together the tool/armor parts that you get from the casting basin. First, sneak-interact (shift-right click) on it until the type of tool/armor you would like to make is displayed on top. Then, interact (right click) on it, and select the tool/armor you want.

Armor Benefits:

Iron - Thorns 2
Gold - Protection 3
Diamond - Protection 4
Netherite - Unbreaking 3

Tool Benefits:

Iron - Unbreaking 2
Gold - Efficiency 4
Diamond - Fortune 3
Netherite - Unbreaking 3

Weapon Benefits:

Iron - Sharpness 3
Gold - Fire Aspect 2
Diamond - Looting 3
Netherite - Unbreaking 3

If the limit of levels is reached on an enchantment, the additional levels are ignored. For example, an iron-iron pickaxe gets unbreaking 3, not unbreaking 4. Axes are considered both a tool and a weapon, so they get both tool and weapon benefits.

Synergy

There is also a benefit to making tools/armor with two part of the same material: Synergy. Any tool or armor that is made up of two pieces of the same material gets the ability to mend itself with experience points. This also applies to netherite-gold tools/armor, as netherite is an alloy containing gold.
